Help Menu

Mod Commands Description
/ban Ban a user from the server
/kick Kick a user from the server
/lockchannel Lock a channel to prevent users from sending messages
/lockedchannels List of locked channels on the server
/mute Mute a user, preventing them from sending messages in the server
/muted List of muted users on the server
/purge Delete a specified number of messages in a channel
/setmodlogchannel Set the mod log channel for the server
/unlockchannel Unlock a previously locked channel
/unmute Unmute a previously muted user
/warn Warn a user on the server
